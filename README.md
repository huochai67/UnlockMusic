# UnlockMusic 音乐解锁
- 解锁加密的音乐文件。

## 支持的格式
- [x] QQ音乐 (.qmcflac/.qmc)
- [x] 网易云音乐格式 (.ncm)

## 使用方法
- 拖拽文件或者文件夹到程序UnlockMusic上
- 或者使用命令行
```
./UnlockMuisc [files.../dirs...]
```
